# 数据库系统小学期第一周周计划 — 王宇航（缓冲池替换策略 + 页保护）

1. 阅读 BusTub 源码中 `buffer/lru_k_replacer.cpp`、`buffer/clock_replacer.cpp` 和 `buffer/page_guard.cpp`，深入理解两种缓冲池替换策略（LRU-K 的 backward k-distance 算法、Clock 的环形扫描机制）以及 RAII 风格的 PageGuard 页保护机制，输出学习笔记和算法伪代码。
2. 定义 `Replacer` 虚基类接口（`Victim` / `Pin` / `Unpin` / `Size`），实现 `ClockReplacer`（环形缓冲 + 引用位指针扫描）和 `LRUKReplacer`（多时间戳追踪 + K 距离计算），编写单元测试验证淘汰正确性、Pin/Unpin 状态转换、边界条件处理。
3. 实现 `ReadPageGuard` 和 `WritePageGuard` RAII 封装（构造获取锁 + 析构释放锁和 Unpin），支持移动语义、禁止拷贝，编写 RAII 正确性、异常安全、多线程并发的单元测试。
4. 与李锦华联调，将 Replacer + PageGuard 接入 BufferPoolManager，编写不同访问模式（顺序扫描 / 80-20 热点 / 均匀随机）下 LRU-K vs Clock 的命中率对比测试，输出选型建议文档。
5. 协助肖清峰执行端到端集成测试（关注缓冲池命中率和 Replacer 行为），协助刘晗阳进行 Benchmark 数据采集，输出 Replacer 模块设计文档（接口定义、算法原理、适用场景对比）。
