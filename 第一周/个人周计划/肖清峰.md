1. 研读 BusTub 的 `execution/executor_factory.cpp` 工厂模式和 `AbstractExecutor` 的 Init/Next 接口设计，研读 MySQL `sql/handler.h` 中 handler 虚函数表与 handlerton 单态机制，输出接口设计文档（handlerton 结构体字段、handler 虚函数表、引擎注册流程），与 B/C/刘晗阳对齐各自需要暴露的 C++ 接口签名。

5. 完成 goods_engine 的 handlerton 注册，编写端到端集成测试（注册引擎 → 建表 → 插入 10000 行 → 全表扫描验证 → 创建索引 → 索引查询验证 → 更新/删除 → 验证一致性 → 删除表 → 注销引擎），输出阶段一验收演示脚本。
