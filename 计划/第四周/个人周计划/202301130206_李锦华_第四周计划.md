# 数据库系统小学期第四周周计划 — 李锦华（备份子系统 + 桌面客户端辅助）

1. 实现 goods_db_dump 全量备份工具——遍历 Catalog 获取所有数据库和表的列表 → 对每张表发送 CREATE TABLE 获取 DDL → SELECT * 获取全部数据 → 生成批量 INSERT 语句（每 100 行一批）→ 写入输出文件。支持 --no-create-info / --no-data / --complete-insert / --single-transaction / --result-file 选项。
2. 实现增量备份功能——基于 Binary Log 位点：记录上次备份的 binlog 文件名和 position → 下次备份从上次位点读取 binlog 事件 → 将 QUERY/ROW 事件转换为 SQL 语句输出。支持 --incremental --since-position 选项。
3. 实现恢复功能——读取 .sql 备份文件 → 分条执行 SQL 进行恢复，支持 --force 选项（遇到错误继续执行）。编写全量备份→恢复→增量备份→恢复的全链路闭环测试（模拟生产环境灾难恢复流程）。
4. 实现备份进度显示（大表备份时输出当前行数/总行数和进度百分比）和备份一致性保证（--single-transaction 选项：REPEATABLE READ 事务内导出保证快照一致性）。
5. 协助刘晗阳实现桌面端备份恢复向导——确保 BackupWizard 的备份操作通过 GoodsDbClient 正确调用底层 dump 功能，恢复操作正确读取备份文件并执行。
6. 输出备份子系统完整设计文档（全量/增量备份流程图 + 恢复流程图 + 备份策略建议 + 操作手册），参与全组最终验收演示（负责备份恢复 Demo）。
