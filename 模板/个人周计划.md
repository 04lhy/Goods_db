# 数据库系统小学期第一周周计划

1. 阅读《数据库系统设计》教学指导手册，了解小学期的任务和计划，根据指导书的内容，确定小学期的目标和方向，先大致确定任务和目标。
2. 确定要阅读的开源数据库系统，下载源码，安装合适的 IDEA 以及尝试调通编译环境。
3. 完成源码的编译，使其正常运行。
4. 参考说明文档以及一些其他的资料，厘清 DBMS 的大致架构，动手尝试增加断点或抛出异常，找到源文件路口，大致对重点部分的掌握有一个阅读记录。
5. 根据老师提供的选题，确定一个自己感兴趣的方向，尝试思考一些优化的方向和思路。并借鉴一些优秀的优化方向。
6. 在对源码有一定的了解后，尝试对一些基本的点进行简单的修改尝试，开始着手写代码。
