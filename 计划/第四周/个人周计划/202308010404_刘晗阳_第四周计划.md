# 数据库系统小学期第四周周计划 — 刘晗阳（组长 / 桌面客户端完善整合）

1. 实现桌面端管理面板 AdminPanel——服务器状态仪表盘（QPS/连接数/缓冲池命中率/脏页数/运行时间实时数值展示 + QTimer 每 5 秒自动刷新）、进程列表 QTableView（ID/用户/主机/数据库/命令/时间/状态 + Kill 按钮）、操作按钮组（Flush Hosts/Logs/Tables + Reload + Ping + Shutdown），与 goods_db_admin 底层命令对接确保功能一致。
2. 实现桌面端备份恢复向导 BackupWizard——QWizard 分步页面：选择操作类型（全量/增量/恢复）→ 选择数据库/表（QTreeWidget 带复选框）→ 选择导出路径/备份文件 → 选项配置（是否导出 CREATE TABLE / 批量 INSERT 行数）→ 执行（进度条 + 实时日志）→ 完成（耗时/文件大小统计），与李锦华的 goods_db_dump 功能对接。
3. 实现桌面端日志查看器 LogViewer——Error Log / Query Log / Binary Log 三个标签切换，每标签含过滤栏（级别/用户/时间范围）+ 内容区（QTableView 或 QPlainTextEdit）+ 搜索栏（关键词高亮），通过服务器端 SHOW LOGS / SHOW BINLOG EVENTS 接口获取数据。
4. 实现桌面端用户管理界面 UserManager——用户列表 QTableView + 新建/删除/修改密码对话框 + 权限编辑区域（QTreeWidget 数据库→表层次 + 8 种权限 CheckBox 勾选矩阵 + 应用权限按钮发送 GRANT/REVOKE）。
5. UI/UX 全面打磨——深色/浅色主题全面测试确保所有控件渲染正常、快捷键冲突检测与修复、窗口布局保存/恢复（QSettings 持久化 Dock 位置和大小）、中英文国际化切换完整覆盖（Qt Linguist .ts 文件 >50 个 UI 字符串）、错误处理体验优化（断连友好提示/自动重连选项/SQL 错误消息）。
6. 组织全组最终验收演示——桌面端完整操作流程 Demo（连接→SQL 编辑→结果浏览→对象树→管理面板→备份向导→日志查看→用户管理→主题切换），汇总全组交付物（代码+测试+文档+配置文件），编写项目总结，提交最终版本。
