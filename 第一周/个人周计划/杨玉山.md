# 数据库系统小学期第一周周计划 — 杨玉山（表与元组层）

1. 阅读 BusTub 源码中 `storage/page/table_page.cpp`，理解 Slotted Page 的页面布局（页头 + slot 数组向前增长 + tuple 数据从页尾向前增长），实现 TablePage 类：InsertTuple / GetTuple / MarkDelete / UpdateTuple，完成页内增删改查和空间回收的单元测试。
2. 阅读 BusTub 源码中 `catalog/column.cpp`、`catalog/schema.cpp` 和 `type/value.cpp`，理解类型系统和行格式设计，实现 Column / Schema / TypeId 枚举 / Value 联合体 / Tuple 序列化与反序列化，特别是支持 VARCHAR 变长字段以适应新零售场景（仓库名、地址、运单号等），完成序列化往返测试和 VARCHAR 边界测试。
3. 阅读 BusTub 源码中 `storage/table/table_heap.cpp` 和 `storage/table/table_iterator.cpp`，实现 TableHeap（双向链表多页容器）的完整 CRUD：InsertTuple → RID、GetTuple(RID)、UpdateTuple（原地或标记删除+末尾插入）、MarkDelete / ApplyDelete，以及 TableIterator 的顺序扫描。完成 10 万行插入→全表扫描→随机更新→随机删除的压力测试。
4. 实现 DDL 表管理操作：CreateTable（创建 `.db` 文件 + 分配首页 + 写入 Catalog）、OpenTable（打开已有文件 + 重建 TableHeap）、CloseTable（刷脏页 + 关文件）、DeleteTable（刷脏页 + 删文件 + Catalog 移除）。
5. 与 肖清峰对接，提供 InsertTuple / GetTuple(RID) / MakeIterator 等 handler 所需接口，输出 Tuple 物理格式说明、RID 结构和页链表图示文档。
