# 数据库系统小学期第三周周计划 — 刘晗阳（组长 / 桌面客户端主导）

1. 搭建 goods_db_studio Qt 6 工程骨架——创建完整目录结构（src/connection/ / sql_editor/ / result_view/ / object_tree/ / network/ / resources/），编写 CMakeLists.txt（find_package Qt6 + AUTOMOC/AUTORCC/AUTOUIC），编写 main.cpp 入口和主窗口框架，配置 .qrc 资源文件（图标/主题 qss/翻译文件），确保 cmake + make 编译通过且应用可启动。
2. 实现 MainWindow 主窗口布局——菜单栏（文件/编辑/查询/工具/设置/帮助 + 快捷键绑定）+ 工具栏（新建连接/打开文件/执行 F5/停止/提交/回滚）+ 中央多标签编辑器 + 左/右/底三个 DockWidget + 状态栏（连接状态/当前数据库/事务状态），实现深色/浅色主题样式表和切换功能。
3. 实现网络客户端库 GoodsDbClient——基于 QTcpSocket 封装 Connect/Disconnect/Ping/Authenticate/Execute 接口，实现 Protocol_text 客户端适配（解析 ColumnCount → ColumnDefinition → RowData → EOF → QueryResult），编写连接/认证/查询/断开全流程测试。
4. 实现 ConnectionDialog（主机/端口/用户/密码/默认数据库/连接名 + 测试连接按钮，QSettings 持久化连接历史）和 ConnectionPool（多服务器连接管理、切换、状态栏实时显示）。
5. 实现 SqlEditorWidget 多标签编辑器——QTabWidget 管理 QPlainTextEdit 标签页 + SqlHighlighter（SQL 语法高亮：关键字/数据类型/字符串/数字/注释/函数 6 类规则）+ 行号区域 + Ctrl+T/W/S/O 快捷键 + 代码补全（QCompleter：关键字 + 表名 + 列名）。
6. 实现 QueryExecutor——F5 执行全部 / Ctrl+Enter 执行当前语句，调用 GoodsDbClient::Execute → 结果集返回后展示在底部结果浏览器；实现查询历史持久化到本地 SQLite（最近 1000 条 + 时间戳 + 耗时）。
7. 实现 ResultTableModel（QAbstractTableModel 绑定查询结果集）+ ResultTableView（NULL 特殊渲染/数值对齐/分页浏览/导出 CSV/JSON/SQL INSERT）+ ObjectTreeWidget（QTreeView 展示服务器→数据库→表→列/索引层次 + 双击查询 + 右键菜单 + 拖拽表名到编辑器）。
