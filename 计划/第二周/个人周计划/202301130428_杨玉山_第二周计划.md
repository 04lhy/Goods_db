# 数据库系统小学期第二周周计划 — 杨玉山（表与元组层 / Schema）

1. 实现 Binder——`Binder::Bind(ASTNode*)` → `BoundStatement*`，遍历 AST 进行语义绑定：表名绑定（AST 表名 → Catalog 查询 → TableInfo）、列名绑定（AST 列名 → Schema 查询 → Column 索引）、类型推导与检查（表达式结果类型、WHERE 条件 BOOLEAN 校验、INSERT 值与列类型兼容性）、聚合函数与 GROUP BY 语义校验（SELECT 中非聚合列必须出现在 GROUP BY 中）。
2. 定义全量 BoundStatement 派生类——BoundSelectStatement / BoundInsertStatement / BoundUpdateStatement / BoundDeleteStatement / BoundCreateStatement / BoundDropStatement / BoundCreateIndexStatement，包含各语句类型所需的全部字段。
3. 实现子查询绑定（递归绑定为子 BoundStatement）、JOIN 绑定（ON 条件的列名在左右表 Schema 中解析）、别名处理（表别名和列别名作用域管理）。
4. 实现 ValuesExecutor——Init 读取 BoundValuesStatement 中的行数据生成 tuple 列表，Next 依次 emit。
5. 实现 DDL Binder 对接——CREATE TABLE 列定义 → Schema 对象构造 + 主键/UNIQUE/NOT NULL 约束识别 + 自动创建主键 B+Tree 索引；DROP TABLE 表名验证；CREATE INDEX 索引类型识别（BTREE/HASH），与 Catalog/TableHeap 联调打通 DDL 执行全链路。
6. 实现 SHOW TABLES 和 DESCRIBE table_name 命令，输出 Binder + Schema 完整设计文档（BoundStatement 类层次图 + Binder 遍历流程 + 类型检查规则表 + 与 Catalog 交互时序图）。
