# goods_db 启动指令

> 每次启动数据库和前端时按顺序执行。需要 **3 个终端**。

---

## 编译

```bash
# 服务端
mkdir -p goods_db/build && cd goods_db/build
cmake .. -DCMAKE_BUILD_TYPE=Debug
cmake --build . -j$(nproc)

# 桌面客户端 + Web 服务器
mkdir -p goods_db_studio/build && cd goods_db_studio/build
cmake .. -DCMAKE_BUILD_TYPE=Debug
cmake --build . -j$(nproc)
```

---

## 终端 1：数据库服务器

```bash
# 先清理上次可能残留的进程
pkill goods_db_server 2>/dev/null; sleep 0.5

cd goods_db/build
rm -rf ./data
./src/sql/goods_db_server --port 3307
```

## 终端 2：Web 前端服务器

```bash
cd goods_db_studio/build
./goods_db_web
```

## 终端 3：加载电商演示数据（可选）

```bash
./goods_db/scripts/setup_demo_data.sh
```

## 浏览器

打开 http://localhost:8080 即可开始可视化操作。

---

## 补充：桌面客户端（可选，代替 Web 前端）

```bash
cd goods_db_studio/build
./goods_db_studio
```

## 补充：命令行管理工具

```bash
# 交互式管理
./goods_db/build/tools/goods_db_admin

# 数据导出
./goods_db/build/tools/goods_db_dump
```
