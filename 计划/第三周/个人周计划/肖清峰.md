# 数据库系统小学期第三周周计划 — 肖清峰（引擎接口层 / 网络协议层）

1. 研读 MySQL 8.0 sql/protocol.h 和 sql/net_serv.cc 源码，设计 goods_db 网络协议包格式规范——Packet Header（payload_length + sequence_id）、Command Packet（command_byte + sql_string）、OK/ERR/ResultSet/EOF Response Packet 的字节级布局定义，输出协议设计文档。
2. 实现 Protocol 抽象基类——store 方法族（store_null / store_integer / store_float / store_string / store_decimal）、send 方法族（send_result_metadata / send_row / send_ok / send_error / send_eof），定义完整虚函数表。
3. 实现 Protocol_text 子类——所有数据按文本行格式序列化（列值 \t 分隔，行 \n 结束），send 方法按包格式封装发送。实现 Protocol_binary 子类（固定长度直接写入 + 变长先写长度）作为高性能备选。
4. 实现 Connection 类——封装 socket 读写（ReadPacket 阻塞读取 → 解析 header+payload / WritePacket 按协议格式发送 / Close），管理连接状态生命周期。
5. 实现 net_serv.cpp 网络服务层——CreateServerSocket (bind+listen) / Accept (blocking accept) / SetNonBlocking (可选非阻塞 IO)，支持多客户端并发连接。
6. 与刘晗阳联调桌面端网络客户端库——确保 GoodsDbClient 通过 Protocol_text 与服务器正确通信（认证 → 查询 → 结果解析 → 断开全链路），输出网络协议层完整设计文档（类层次图 + 连接状态机 + 协议包规范 + 与 MySQL 协议差异对照表）。
