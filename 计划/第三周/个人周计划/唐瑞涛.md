# 数据库系统小学期第三周周计划 — 唐瑞涛（项目骨架 / 工具链 / CI）

1. 实现 ThreadPool 线程池——预创建 N 个 worker 线程（N 可配置），线程安全任务队列 ConcurrentQueue（基于 readerwriterqueue），Submit(task) 返回 future<Result>，支持优雅关闭（等待任务完成）和强制关闭（丢弃未执行任务）。
2. 实现 goods_db 服务器启动流程 main()——解析配置 → 初始化 Logger → 注册存储引擎 → 初始化 BufferPoolManager → 加载 Catalog → 启动 ThreadPool → 创建网络监听 socket → accept 循环接收连接，每连接 Submit 到 ThreadPool。
3. 实现 ConnectionHandler 连接处理器——管理单个客户端连接的完整生命周期：接收数据包 → 解析命令类型（QUERY/PING/QUIT）→ 认证（调用 check_connection）→ 执行 SQL（调用 Execute）→ 序列化结果集 → 发送响应 → 清理资源。维护连接状态机（INIT → AUTH → READY → QUERYING → SENDING → READY / CLOSING）。
4. 实现服务器优雅关闭——SIGTERM/SIGINT 时：停止 accept 新连接 → 等待所有活跃连接完成 → FlushAllPages → 关闭所有文件 → 退出。编写完整的启动/关闭/重启集成测试。
5. 服务器整体性能调优——ThreadPool worker 最优数量测试、socket 参数调优（TCP_NODELAY/SO_RCVBUF）、epoll IO Multiplexing 优化（可选扩展）。
6. 更新 CI 流水线——增加网络层/日志层编译目标 + 多客户端并发集成测试 + Qt 6 依赖安装，输出服务器部署文档（启动命令 + 配置参数完整清单 + systemd 服务模板 + 性能调优推荐值表）。
